Learned till arrays from MDN therefore decided to make a Calculator using ARRAYS <br>
<h1> How does this works? </h1> <br>
First the user enters a number then they click on add second number after clicking the add Second number option the JS dumps the first number from first Array to second array So now the first array is empty the user again enters the number and clicks on add,sub,multiply or divide to do the calculations <br>
I have also tried to implement animations on click as it makes the website more interactive
